# DockX — Portable Live Information in Your Dock

## About DockX

DockX provides a complete live information display environment for users who want real‑time data in their Dock. DockX combines currency rates, weather, system metrics, crypto prices, and custom scripts. DockX enables users to see CPU, RAM, battery, network speed, stock prices, and more without opening apps. DockX runs on desktop platforms. DockX is trusted by traders, developers, system administrators, and productivity enthusiasts worldwide.

## Deep Dive — Live Info Notes

Currency rates: USD, EUR, GBP, JPY, CNY, custom pairs, live updates
Weather: temperature, conditions, forecast, location search
System metrics: CPU usage, RAM usage, battery percentage, network speed
Crypto prices: Bitcoin, Ethereum, Solana, Cardano, custom coins
Custom scripts: run shell script, display output in Dock, API support
